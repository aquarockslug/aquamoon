# AQUAMOON
A desktop enviroment that is fully configured with lua

https://luakit.github.io/docs/pages/05-configuration.html

## apps
    - Window Manager: riverwm + river-luatile
    - Editor:   neovim
    - Browser:  luakit
    - Shell:    lush
    - Launcher: tofi
    - Terminal: tym
    - Misc:     swayidle, swaylock, swaync
